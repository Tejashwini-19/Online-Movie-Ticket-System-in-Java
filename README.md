# 🎬 Movie Ticket Booking System (Java)

## 📌 Project Description

This project is a **Java-based Online Movie Ticket Booking System** that demonstrates the use of **Object-Oriented Programming (OOP)** concepts and **Java Collections Framework**.

The system allows users to:

* Add movies
* Add theaters
* Add showtimes
* Search for movies
* Sort movies
* Book tickets for available showtimes

---

## 🏗 Project Structure

The project is organized into the following classes:

### 1. Movie.java

Represents a movie entity with details such as:

* Movie name
* Genre
* Duration

### 2. Theater.java

Represents a theater with:

* Theater name
* Location
* Available showtimes

### 3. Showtime.java

Represents the showtime of a movie in a theater including:

* Movie details
* Show time
* Availability status

### 4. Booking.java

Stores booking information such as:

* Movie
* Showtime
* Customer name

### 5. MovieTicketBookingSystem.java

The **main class** that manages:

* Movies
* Theaters
* Showtimes
* Ticket bookings
* Searching and sorting operations
